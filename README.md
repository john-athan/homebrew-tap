# homebrew-tap

[John Athan](https://github.com/john-athan)'s personal Homebrew tap.

```bash
brew install john-athan/tap/<formula>
```

## Formulae

| Formula | What it is |
|---|---|
| [`taurine`](https://github.com/john-athan/taurine) 🐂 | Keep your Mac awake, with a reason — menu bar caffeine tool. |
| [`sucher`](https://github.com/john-athan/sucher) | Fast terminal viewer for markdown, spreadsheets, PDF, images, video & docx. |

```bash
brew install john-athan/tap/taurine
brew install john-athan/tap/sucher
```

Formulae here build from source on your machine, so there's no Gatekeeper
"unidentified developer" prompt and nothing to notarize.
